GINIUserCenterManager

@interface GINIUserCenterManager : NSObject

The GINIUserCenterManager handles the communication with the Gini User Center via the Gini User Center API (see http://developer.gini.net/gini-api/html/user_center_api.html for details). It can be used to create new users, log in a user or to get information on a user.

Warning

Access to the User Center API is restricted to selected clients only.
