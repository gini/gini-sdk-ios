GINISessionManager

@interface GINISessionManager
    : NSObject <GINISessionManager, GINIIncomingURLDelegate> {
  NSString *_clientID;
  NSURL *_baseURL;
  NSString *_appScheme;
  id<GINIURLSession> _URLSession;
  GINISession *_activeSession;
  BFTaskCompletionSource *_activeLogInTask;
  NSString *_activeLogInState;
}

The session manager is responsible for the authorization against the GINI server whenever is needed. authorization is required to perform requests to the GiniAPI.

See

http://developer.gini.net/gini-api/html/guides/oauth2.html
